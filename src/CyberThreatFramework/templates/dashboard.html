Operator's Manual

Welcome to ThreatSimPro

This framework is a professional Cyber Threat Simulation tool designed to model, execute, and analyze cyber attacks in a controlled environment. It helps security professionals understand attack vectors and validate defense mechanisms.

How to Use
1. Monitor Status

Check the Dashboard for global threat metrics, recent activity logs, and the visual attack chain.

2. Execute Simulations

Go to the Simulations tab. Select an attack vector (e.g., Ransomware, PLC Attack) and a target IP. Click "Execute" to start the attack engine.

3. Analyze Results

Watch the Live Terminal for real-time feedback. Once complete, a "Compromise Detected" report will appear with remediation steps.

System Architecture

Frontend: Web Dashboard (HTML/JS/Bootstrap)
Backend: Flask Server (Python)
Engine: Modular Python Scripts (Scapy, Paramiko, Cryptography)
Visualization: Matplotlib & NetworkX

Note: All attacks are simulated safely. "Ransomware" only encrypts files in a dummy directory.
